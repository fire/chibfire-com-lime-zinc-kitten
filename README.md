## Getting started on Windows 11

```
cmd
"C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\VC\Auxiliary\Build\vcvars64.bat"
mix deps.get
scoop install cmake
set LIBTORCH_TARGET=cu111
mix compile
iex.bat --werl -S mix
```
